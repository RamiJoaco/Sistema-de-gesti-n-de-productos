# Sistema-de-gestión-de-productos

Este primer proyecto en python se hizo con la práctica de programacion basica.

Funciones del programa:

Mostrar productos
Buscar productos
Agregar productos
Eliminar productos
Salir del sistema utilizando el menú interactivo.

Las herramientas utilizadas en el proyecto fueron:
Lista y sublistas
Bucles While y For
Condicionales if, elif y else
Variables
Metodos .append() y .pop()

El objetivo del proyecto, fue practicar python, lo aprendido en clases y del material teorico dictado por Talento Tech
